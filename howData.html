How Data Player Works?

At first glance you will see a candlestick with the tittle of EURUSD, and a counter indicating 99 Candlesticks.

Also in left side there are a few inputs, and two sliders in the bottom of the screen;

But What’s going on with all this stuff?

Data player has the function of reproduce price movements of the Forex currencies in only one candlestick simply by press the play button.

The tittle pair indicates the currency pair that is currently loaded.

When you press play button you will see how the EURSD prices go until the last price given by Alpha Vantage

You can change the pair using the Pair selectors “From” and “To”, and then press the submit button to load the candlesticks related to the pair selected.

Each frame of the movement is formed by default with close prices of 60 min interval, but you can change this interval using the Time Frame selector and then pressing the submit button.

The counter indicates the number of prices loaded to animate our candlestick, by default 100 prices are going to load, but if you want more history prices you can check on the Full Data Size box before clicking the submit button.

Also you will see a slider moving on, that represents the point where the data goes. You can drag the slider wherever you want to reproduce a specific part of the whole movement.

By checking the Full data size checkbox the API will return the full-length time series.

The smaller slider is for the speed, it controls how many time each frame will hold down before the next price starts to show.
